# PSWindowsTimeSyncRepair

A PowerShell script to repair Windows time synchronization issues.

## Description

This script checks for common issues that prevent Windows from synchronizing time correctly, including DNS configuration problems and time service settings.

## Prerequisites

- Windows operating system
- PowerShell (comes with Windows)
- Administrator privileges (the script will request elevation)

## Usage

1. Download `timeSyncFix.ps1`
2. Right-click the file and select "Run with PowerShell" (as Administrator)
3. Follow the on-screen instructions
